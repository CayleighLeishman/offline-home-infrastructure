### Commands I Used and Learned

| Command                                         | The Analogy              | Technical Function                                                                            |
| :---------------------------------------------- | :----------------------- | :-------------------------------------------------------------------------------------------- |
| **`apt install`**                               | **Ordering Supplies**    | Downloads and sets up new tools (like Samba) to bring them into the system.                   |
| **`apt update`**                                | **Checking the Catalog** | Refreshes the list of available software versions to see what can be downloaded.              |
| **`ifconfig`**                                  | **The Signpost**         | Displays network interface details and your "Street Address" (IP) so devices can find you.    |
| **`ip addr show`**                              | **The Address Book**     | Shows all IP addresses and interfaces on your machine, similar to `ifconfig` but more modern. |
| **`service [name] restart`**                    | **The Reset Button**     | Forces a background worker (Service) to stop and start over to apply new rules.               |
| **`smbpasswd -a`**                              | **The Staff Badge**      | Creates secure credentials (passwords) so authorized users can access shared folders.         |
| **`sudo`**                                      | **The Master Key**       | Grants "SuperUser" (Admin) permissions to perform restricted system tasks.                    |
| **`vim`**                                       | **Picking up the Pen**   | Opens the text editor to modify "Rulebooks" (configuration files) and set system laws.        |
| **`ssh-keygen -t ed25519`**                     | **Magic Padlock Maker**  | Generates a public/private key pair for secure login to remote devices.                       |
| **`sudo sysctl -w net.ipv4.ip_default_ttl=65`** | **TTL Tuner**            | Adjusts the Time-To-Live value on packets to bypass certain network restrictions.             |
| **`ping <Target_IP_Address>`**                  | **The Echo Shout**       | Sends ICMP requests to see if a device is alive and reachable.                                |
| **`ip -6 addr`**                                | **The IPv6 Directory**   | Lists all IPv6 addresses assigned to your interfaces.                                         |
